Para crear un repositorio local entramos en la carpeta con los fuestes:

	git init

opcionalmente ponemos el nombre de la carpeta o proyecto y git creará la carpeta raiz

	git init <nombre proyecto>
	

Para ver el estado del proyecto:
	
	git status
	
Para añadir un archivo:

	git add <nombre del archivo>
